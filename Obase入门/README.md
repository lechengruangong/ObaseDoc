本文件夹的内容为Obase的入门内容,如果是没有接触过Obase的用户,请按照如下的顺序阅读以下四篇文档:
1. [快速开始(dotNet版)](./快速开始_dotNet.md)或者[快速开始(java版)](./快速开始_java.md).
2. [最佳实践(dotNet版)](./最佳实践_dotNet.md)或者[最佳实践(java版)](./最佳实践_java.md).
3. [深入理解(dotNet版)](./深入理解_dotNet.md)或者[深入理解(java版)](./深入理解_java.md).
4. [代码优先(dotNet版)](./代码优先_dotNet.md)或者[代码优先(java版)](./代码优先_java.md).

选取这些文档中你使用的语言平台的版本阅读即可.

在对Obase有了初步的了解后,以下四篇文档则没有固定的阅顺序,选择你想要优先了解的内容阅读即可.

- [Obase如何定义和管理对象上下文(dotNet版)](./Obase如何定义和管理对象上下文_dotNet.md)和[Obase如何定义和管理对象上下文(java版)](./Obase如何定义和管理对象上下文_java.md).主要介绍了Obase的对象上下文相关的内容,包含如何定义和使用上下文两个部分.
- [Obase如何定义和管理对象上下文配置提供器(dotNet版)](./Obase如何定义和管理对象上下文配置提供器_dotNet.md)和[Obase如何定义和管理对象上下文配置提供器(java版)](./Obase如何定义和管理对象上下文配置提供器_java.md).主要介绍了Obase的对象上下文配置提供器相关的内容,包含如何定义和使用上下文的配置提供器两个部分.
- [Obase如何配置对象数据模型(dotNet版)](./Obase如何配置对象数据模型_dotNet.md)和[Obase如何配置对象数据模型(java版)](./Obase如何配置对象数据模型_java.md).主要介绍了Obase的对象数据模型配置相关的内容,包含配置对象数据模型思路和对象数据模型配置示例两个部分.
- [Obase如何进行查询(dotNet版)](./Obase如何进行查询_dotNet.md)和[Obase如何进行查询(java版)](./Obase如何进行查询_java.md).主要介绍了Obase支持哪些查询方法和这些查询方法会映射为哪些关系运算.
